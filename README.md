StockTrack
==========
Application to Showcase XML Services using Google Stock API.


# Purpose
The purpose of this application is to showcase XML service of Google Stock API.

# Supported platforms:
**Mobile**
 * Android
 * iPhone
 * BlackBerry
 * SPA Android
 * SPA iPhone
 
**Tablet** 
 * iPad
 * Android Tablet
 * SPA iPad
 * SPA Android Tablet
 
**Desktop**
 * DesktopWeb
